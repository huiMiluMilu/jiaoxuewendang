## 删除时怎么删最稳

删除时请删**真实目录**，不要删旧的短链软链接。

例如：

```bash
rm -rf '/Users/shihui/Desktop/玩玩编程/生产教学文档/simple-demo/第七节课-爆款心理测试与社媒传播逻辑-hVGcxc'
```

如果你不确定一个目录是不是软链接，可以先看：

```bash
ls -la '/Users/shihui/Desktop/玩玩编程/生产教学文档/simple-demo'
```

带 `->` 的就是软链接，不是主要目录。
